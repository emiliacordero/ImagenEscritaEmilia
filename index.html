Encargos clase a clase

s Luego de cada clase se debían realizar ejercicios en el programa p5js, los cuales debían cumplir con lo pedido según lo visto en cada una de las clases realizadas. Es por eso que cada ejercicio expuesto en esta página hace referencia a cada encargo realizado semanalmente, durante el segundo semestre del 2021.

Encargos

Encargo 1 Para el primer encargo se hizo un acercamiento al programa p5js. Para lograr esta familiarización con el programa se basó en los ejemplos presentados en el libro "Introducción a P5. js" de Lauren McCarthy, Casey Reas, Ben Fry. Mover mouse para interactuar con el ejercicio.


Encargo 2 Como segundo encargo se pidió realizar un ejercicio en el programa p5js basado en la obra de Sol Lewitt y representarla por medio de códigos aprendidos en clase.


Encargo 3 Para este tercer encargo se desarrolló un ejercicio agregando una variable a lo que se venía trabajando, la cual fue slider, con ella se puede modificar el tamaño del círculo en cuestión. Deslizar hacia izquierda y derecha para modificar el tamaño.


Encargo 4 En este encargo número cuatro se pidió realizar la figura del triángulo de Serpinski, de la forma y color que cada uno eligiera.


Encargo 5 Como encargo cinco se debía realizar un ejercicio de recursividad en el que se proyectara la definición explicada en clases.


Encargo 6 Para este encargo se debía crear una página personal en github, con las condiciones explicadas en clase, con sus debidos nombres y repositorios.


Encargo 7 Como encargo siete se hizo presente la función void, la cual fue realizada en base a lo explicado en clase con sus respectivas propiedades.


Encargo 8 Para este encargo se realizó un ejercicio llamado pizarra interactiva, él cual tuvo como base los códigos entregados por el profesor de Imagen Escrita. Este código fue editado y cambiado con respecto a cada estudiante. Arrastrar mouse y hacer click para interactividad.


Encargo 9 Como encargo número nueve se realizó un ejercicio en el mismo tiempo de la clase el cual debía cumplir con los parámetros explicados. Estos eran realizar ciertos clicks para hacer aparecer figuras y que estas puedan cambiar al tercer click. Hacer click para interactuar con el ejercicio.
